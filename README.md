t - task manager
=================

t is a simple command-line task manager written in Ruby. Inspired by [t-](http://www.penzba.co.uk/t-/t-.html?HN1) which was written in Python.


Usage:
-------

* List todo tasks
       t
* List all tasks
       t -a
* List done tasks
       t -d
* Mark a task as done
       t -m <key>
* Remove a task
       t -r <key>
* Clear task list
       t -c
